Convolution 연산 : mask 내의 원소값과 공간 영역에 있는 입력 영상의 화소값들을 곱하여 출력 계산  
=> 이 과정을 모든 출력 화소값에 대해 이동하면서 수행  
=> 이때 곱해지는 mask를 kernel, window 또는 filte라고도 함
